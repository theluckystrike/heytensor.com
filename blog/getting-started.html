Getting Started with Deep Learning

Your complete roadmap from zero to building and training neural networks with PyTorch.

Setting Up Your Environment

The first step in your deep learning journey is setting up a development environment. You need Python 3.8 or later, PyTorch, and a few supporting libraries. If you have an NVIDIA GPU, install CUDA-enabled PyTorch for dramatically faster training. If not, CPU-based training works perfectly fine for learning and small projects, and free GPU access is available through Google Colab.

# Create a virtual environment
python -m venv dl-env
source dl-env/bin/activate

# Install PyTorch (CPU version for getting started)
pip install torch torchvision torchaudio

# Install supporting libraries
pip install matplotlib numpy jupyter

# Verify installation
python -c "import torch; print(f'PyTorch {torch.__version__}')"
python -c "import torch; print(f'CUDA available: {torch.cuda.is_available()}')"

Understanding Tensors

Before building neural networks, you need to be comfortable with tensors. A tensor is simply a multi-dimensional array, the fundamental data structure in deep learning. Think of a 1D tensor as a list of numbers, a 2D tensor as a table (matrix), and a 3D tensor as a stack of tables. Here are the essential operations you will use constantly:

import torch

# Creating tensors
x = torch.tensor([1.0, 2.0, 3.0])          # From a list
y = torch.zeros(3, 4)                       # 3x4 matrix of zeros
z = torch.randn(2, 3, 4)                    # Random 2x3x4 tensor

# Essential operations
a = torch.randn(32, 784)                    # Batch of 32 flattened images
w = torch.randn(784, 128)                   # Weight matrix
b = torch.randn(128)                        # Bias vector
output = a @ w + b                          # Linear transformation

# Reshaping
images = torch.randn(32, 1, 28, 28)        # Batch of grayscale images
flat = images.view(32, -1)                  # Flatten to (32, 784)
print(flat.shape)                           # torch.Size([32, 784])

Building Your First Neural Network

With tensor basics covered, let us build a simple neural network that classifies handwritten digits from the MNIST dataset. This network has two hidden layers with ReLU activation and a final layer that outputs probabilities for each of the 10 digit classes. The complete implementation, including data loading, model definition, training loop, and evaluation, fits in under 60 lines of code:

import torch
import torch.nn as nn
from torchvision import datasets, transforms
from torch.utils.data import DataLoader

# Load MNIST data
transform = transforms.Compose([
    transforms.ToTensor(),
    transforms.Normalize((0.1307,), (0.3081,))
])
train_data = datasets.MNIST('./data', train=True, download=True, transform=transform)
test_data = datasets.MNIST('./data', train=False, transform=transform)
train_loader = DataLoader(train_data, batch_size=64, shuffle=True)
test_loader = DataLoader(test_data, batch_size=1000)

# Define the neural network
class DigitClassifier(nn.Module):
    def __init__(self):
        super().__init__()
        self.network = nn.Sequential(
            nn.Flatten(),
            nn.Linear(784, 256),
            nn.ReLU(),
            nn.Linear(256, 128),
            nn.ReLU(),
            nn.Linear(128, 10)
        )

    def forward(self, x):
        return self.network(x)

model = DigitClassifier()
optimizer = torch.optim.Adam(model.parameters(), lr=0.001)
criterion = nn.CrossEntropyLoss()

The Training Loop

Training a neural network follows a simple loop: forward pass to compute predictions, loss computation to measure error, backward pass to compute gradients, and optimizer step to update weights. This cycle repeats for every batch in the dataset, and the full dataset is processed multiple times (epochs). Here is the training loop for our digit classifier:

# Training loop
for epoch in range(5):
    model.train()
    for batch_idx, (data, target) in enumerate(train_loader):
        optimizer.zero_grad()           # Clear previous gradients
        output = model(data)            # Forward pass
        loss = criterion(output, target) # Compute loss
        loss.backward()                 # Backward pass (compute gradients)
        optimizer.step()                # Update weights

    # Evaluate after each epoch
    model.eval()
    correct = 0
    with torch.no_grad():
        for data, target in test_loader:
            output = model(data)
            pred = output.argmax(dim=1)
            correct += pred.eq(target).sum().item()

    accuracy = 100. * correct / len(test_data)
    print(f'Epoch {epoch+1}: Test accuracy = {accuracy:.1f}%')

After just 5 epochs of training, this simple network achieves approximately 97 percent accuracy on handwritten digit classification. This is a remarkable result for a model with only three layers, and it demonstrates how powerful even basic neural networks are at learning patterns from data.

Next Steps

With your first neural network working, the natural progression is to explore convolutional neural networks for image tasks, recurrent networks for sequential data, and eventually transformer architectures for the most demanding problems. Each architecture introduces new concepts, but they all build on the same fundamental loop of forward pass, loss computation, and gradient-based optimization. For applied ML practices that complement your deep learning skills, ML3X covers production deployment and feature engineering strategies that take models from notebooks to real-world applications.

As you tackle more complex problems, invest time in understanding how to diagnose training issues: learning rate too high (loss diverges), learning rate too low (loss decreases very slowly), overfitting (training accuracy much higher than test accuracy), and underfitting (both accuracies are low). Each diagnosis has specific remedies covered in our guides section. The ability to diagnose and fix training problems efficiently is what separates effective deep learning practitioners from those who rely solely on trial and error.
